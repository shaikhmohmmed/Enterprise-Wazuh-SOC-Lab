# Enterprise SOC Home Lab using Wazuh SIEM

## Overview

This project demonstrates the deployment of a Security Operations Center (SOC) home lab using Wazuh SIEM in a VirtualBox environment. The objective was to monitor a Windows 10 endpoint, generate security events, analyze alerts, and gain hands-on experience with security monitoring and incident investigation.

---

## Objectives

- Deploy Wazuh Manager using the official Wazuh OVA.
- Install and configure a Windows 10 Wazuh Agent.
- Monitor Windows security events.
- Detect authentication failures.
- Monitor file integrity.
- Detect PowerShell and account discovery activities.
- Monitor user account creation and modification.
- Investigate alerts generated by Wazuh.

---

## Lab Architecture

```
                  VirtualBox

        +-----------------------+
        | Ubuntu (Wazuh OVA)    |
        |-----------------------|
        | Wazuh Manager         |
        | Wazuh Dashboard       |
        | Wazuh Indexer         |
        +----------+------------+
                   |
                   |
         Wazuh Agent Communication
                   |
        +----------+------------+
        | Windows 10            |
        | Wazuh Agent           |
        +-----------------------+
```
